## Getting Started

### Dependencies

Node v14.16.0

    "axios": "^0.21.1",
    "body-parser": "^1.19.0",
    "bulma": "^0.9.2",
    "ejs": "^3.1.6",
    "express": "^4.17.1",
    "jquery": "^3.6.0",
    "math": "0.0.3",
    "moment": "^2.29.1",
    "mongoose": "^5.12.14"
    
    

```
npm install express ejs body-parser axios math moment jquery mongoose bulma  axios
npm install -D nodemon 
```


### Executing program

* How to run and start the app

```
npm install
npm start
```
http://localhost:3000/
